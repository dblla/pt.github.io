Porque Html e Css não são considerados Linguagens de Programação

Embora possa parecer, HTML e CSS não são linguagens de programação.

HTML é uma linguagem de marcação, que serve para descrever aos servidores qual é o estilo e estrutura de um documento, enquanto CSS é uma linguagem de estilo, que serve para estilizar os elementos de documentos escritos em uma linguagem de marcação, CSS significa Cascading Style Sheets (Folha de Estilo em Cascatas). Ele tem esse nome porque aplica-se as estruturas de “cima para baixo”, ou seja, por meio de regras com prioridades para seguir. O recurso é facilmente aplicado com linguagens de marcação.
