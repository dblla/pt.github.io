Diferença entre Java e JavaScript

Mesmo aparentando ter o mesmo nome, elesm são diferentes. Java e Javascript tem sim as suas diferenças.

Java é usado em diversas aplicações, incluindo programação de operadoras de cartão de crédito, aplicativos Android, desktop e apps corporativos. Para efeito de comparação, o JavaScript é usado principalmente para tornar as páginas de aplicativos da web mais interativas.
